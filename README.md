# End4 Hyprland Config

这是一个用于 **End4** 窗口管理器和 **Hyprland** 配置的仓库。包含了各种自定义设置，用于优化界面、动画效果、UI 主题等。

## 内容

### 1. Hyprland 窗口管理器配置
- **general.conf**: 外观和动画设置，包括圆角、模糊效果、阴影、贝塞尔曲线等。
- **rules.conf**: 图层规则和 UI 动画配置。

### 2. Quickshell UI 主题系统
- **Appearance.qml**: 主题和动画曲线配置。
- **Config.qml**: 字体、透明度和壁纸主题配置。

## 安装与使用

### 安装步骤

1. 克隆此仓库到本地：
    ```bash
    git clone https://github.com/xibielemn/End4-Hyprland-Config.git
    cd End4-Hyprland-Config
    ```

2. 将文件复制到相应的配置目录中：
    ```bash
    cp dots/.config/hypr/hyprland/* ~/.config/hypr/hyprland/
    cp dots/.config/quickshell/ii/modules/common/* ~/.config/quickshell/ii/modules/common/
    ```

3. 根据需要修改配置文件，以适应你的系统和需求。

### 配置说明

- **general.conf**：配置 Hyprland 窗口管理器的外观与动画效果，例如窗口圆角、模糊、阴影等。
- **rules.conf**：设置图层规则以及 UI 动画效果，适用于 Quickshell 组件的透明度和模糊效果等。
- **Appearance.qml** 和 **Config.qml**：Quickshell 主题和 UI 动画曲线的配置。

## 特性
- 自定义窗口装饰、动画和工作区效果。
- 适用于 **Hyprland** 和 **Quickshell** 配置。
- 提供了一些实用的 UI 调整，提升用户体验。
